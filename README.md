## **一、先搞清楚搬瓦工是什么**

搬瓦工的英文名是 BandwagonHost，母公司是 IT7 Networks，一家在加拿大注册的老牌主机商。它最早靠低价 KVM VPS 在国内圈子火起来，后来一路把线路往中高端做，现在已经是国内用户做建站、科学上网、跨境业务时绕不开的一个名字。

它有几个比较硬的特点：

- 全部采用 KVM 虚拟化，自研 KiwiVM 控制面板，支持一键换系统、快照、迁移机房
- 自有硬件、自有 IP 段，不租借第三方设备
- 19 个全球机房可选，包括洛杉矶 DC6/DC9、香港、东京、大阪、新加坡、迪拜、纽约、荷兰等
- 支持支付宝、PayPal、信用卡付款，对国内用户友好
- 30 天退款政策，99.9% 在线率承诺

> 简单说，它不是最便宜的 VPS，但它是"价格和线路质量平衡得比较好"的那一类。你花的钱里有相当一部分买的是回程线路，而不是单纯的硬件配置。

## **二、搬瓦工购买前必须搞懂的三件事**

在掏钱之前，有三个概念如果你搞不清楚，大概率会买错。

### **1. 线路类型：CN2 GIA、CN2 GT、普通 163 到底差在哪**

这是搬瓦工套餐之间最大的分水岭。搬瓦工官方对这块的解释挺技术化的，我用大白话翻译一下：

- **普通 163 线路（AS4134）**：最便宜，也是最常见的国际出口。晚高峰容易堵，丢包率可能飙到 30% 以上。适合预算极紧、对延迟不敏感的场景。
- **CN2 GT（AS4809，全球传输）**：本来是来解决 163 拥堵的，但 2019 年之后基本也堵得差不多了，性价比反而尴尬。
- **CN2 GIA（AS4809，全球互联网接入）**：电信最贵的线路，每兆带宽成本能到 120 美元。稳定、低丢包、低延迟，是建站、视频会议、游戏联机的首选。
- **CTGNet（AS23764）**：电信新推出的线路，官方说性能和 CN2 GIA 基本等效。

搬瓦工的套餐命名里，凡是带 **"CN2 GIA-E"**（也叫电子商务套餐）的，就是走 CN2 GIA 回程的高端款；带 **"CN2 GIA"** 但不带 E 的香港、东京套餐，是更顶配的直连款；而最便宜的 KVM 套餐，走的就是普通线路。

### **2. 机房选择：不是越近越好**

很多人觉得香港最近，一定最快。这话对一半——香港延迟确实最低，但它带宽小、流量少、价格贵。如果你是做建站、跑流量大的业务，洛杉矶 DC6 CN2 GIA-E 反而是更划算的选择，因为带宽给到 2.5Gbps，流量也是 1TB 起步。

目前搬瓦工主流机房按线路质量排序，大致是这样：

1. **香港 CN2 GIA**：延迟最低，三网直连，价格最贵
2. **东京 CN2 GIA / 大阪 CN2 GIA**：延迟次低，适合日本方向业务
3. **新加坡 CN2 GIA**：东南亚方向的新选择，2026 年 5 月才上线
4. **洛杉矶 DC6 CN2 GIA-E / DC9 CN2 GIA**：性价比之王，三网回程 CN2 GIA
5. **JPOS_1（日本软银）、EUNL_9（联通荷兰）**：CN2 GIA-E 套餐可迁移的备选机房
6. **DC3 CN2、DC8 ZNET、DC2 QNET 等**：普通线路，价格便宜

### **3. 计费周期：年付永远比月付划算**

搬瓦工所有套餐都是周期越长越便宜。同一个 CN2 GIA-E 1GB 套餐，季付 49.99 美元，年付 169.99 美元——年付相当于打 85 折。如果你确定要用一年以上，直接年付是最优解。

## **五、搬瓦工购买流程：从注册到付款十分钟搞定**

具体步骤如下：

1. **选套餐**：点上面表格里的购买链接，进入对应套餐的下单页
2. **选周期和机房**：在 Billing Cycle 里选年付（划算），在 Location 里选你想要的机房
3. **填优惠码**：在 Promotional Code 框输入优惠码（如果当时有的话），点 Validate Code 验证
4. **注册账号**：填邮箱、密码、个人信息，地址要写真实地址，乱填可能被风控
5. **选支付方式**：国内用户选 Alipay（支付宝）最方便，也支持 PayPal 和信用卡
6. **付款**：扫码支付，完成后 VPS 几分钟内开通
7. **登录 KiwiVM**：在后台 Services 里找到你的 VPS，点 KiwiVM 面板进系统管理

整个过程官方说不超过十分钟，实测确实差不多。唯一要注意的是注册时邮箱建议用 Gmail 或者国外邮箱，国内邮箱偶尔会收不到开通邮件。
